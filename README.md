## Setup

### 1: Install NPM
https://nodejs.org/en/

### 2: Install Gulp
- `npm install`

This will run the package.json file and download the list of modules to a "node_modules" folder in the plugin.

### 3: Gulp Commands
- `gulp watch`
- `gulp compile-css`
- `gulp compile-js`
- `gulp wordpress-lang`
